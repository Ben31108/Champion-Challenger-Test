# Statistical Significance Tool

This project implements a statistical testing tool in Python to evaluate whether differences between two proportions are statistically significant.

The tool is designed for practical business applications such as A/B testing and campaign analysis, allowing users to make data-driven decisions with confidence.

Key Features:
• Hypothesis testing for proportions
• Clear interpretation of results
• Designed for marketing and experimental analysis use cases

Tools: Python (Statsmodels, NumPy)

Caveat: I wrote the original program using the Sublime editor, and have executed it from that editor and from Jupyter Notebooks. To run it the user will require an environment that supports Tkinter.
